Open Source

Welcome to the home of eBay’s open source projects. Many of the projects you will find here started off as internal eBay projects, or as personal projects of our employees. We hope that you will be able to incorporate these projects into your own work, and we strongly encourage you to participate in our developer community.

Featured Project - Fabio

Fabio is a fast, modern, zero-conf load balancing HTTP(S) router for deploying microservices managed by consul.

Learn more

Featured Project - Apache Kylin

Apache Kylin is an open source Distributed Analytics Engine designed to provide SQL interface and multi-dimensional analysis (OLAP) on Hadoop supporting extremely large datasets.

Learn more

Featured Project - Apache Eagle

Apache Eagle is an Open Source Monitoring solution to instantly identify access to sensitive data, recognize attacks, malicious activities in Hadoop and take actions in real time.

Learn more

More Projects

Some additional projects are found below. You can find a full list of our projects on www.github.com/ebay. Open source is supported by the very top levels of the company, so expect to see many more projects from eBay!

Pulsar

Pulsar is an open-source, real-time analytics platform that includes stream processing, metrics store, and reporting frameworks. Pulsar can be used to collect, process user and business events in real time, provide key insights using custom dashboards, and enable systems to react to user activities within seconds.

Learn more

Parallec

Parallec is a fast parallel async HTTP(S)/SSH/TCP/Ping client java library based on Akka. Scalably aggregate and handle API responses anyway and send it anywhere by writing 20 lines of code.

Learn more

NMessenger

NMessenger is a fast, lightweight messenger component built on AsyncDisplaykit and written in Swift. Developers can inherently achieve 60FPS scrolling and smooth transitions with rich content components.

Learn more

UAF

Universal Authentication Framework. The main goal is the passwordless authentication experience

Learn more

Gzynga

Seekable and Splittable GZip. This project is aim to provide Seekable (random search within gzip file) and Splittable (can divide gzip file into multiple chunks) capability for gzip compressed file.

Learn more

Sherlock

Sherlock provides real time monitoring of our infrastructure, services and applications.
